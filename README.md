# 🧙 WizardVerse RPG Campus System

An interactive **RPG-style educational web world** built using HTML, CSS, and JavaScript.  
Students explore magical locations, play mini-games, earn XP, level up, and track achievements like a real game.

---

## 🌍 Live Concept

WizardVerse transforms a normal campus website into a **fully interactive RPG universe**:

- Explore different magical locations
- Complete mini-games to earn XP
- Level up your wizard character
- Track achievements in the Great Hall
- Navigate using Map Tower

---

## 🎮 Game World

### 🌲 Enchanted Forest
- Treasure hunt game
- Random rewards
- Exploration experience

### 🧪 Potion Lab
- Innovation & research hub
- Educational mini-systems

### 📚 Grand Library
- Knowledge center
- Study and learning interface

### 🏛 Magic Hall
- Learning and spell-based challenges

### 🗺 Map Tower
- Navigation system
- Fast travel between locations

### 🏰 Castle
- Central administration
- RPG battle system (Wizard Chess, Spell Challenge)

### 🏆 Great Hall
- Shows XP, level, and achievements
- Player progress dashboard

---

## ⚙️ Features

✔ XP System (stored using localStorage)  
✔ Level System (Novice → Legendary Wizard)  
✔ Achievement System  
✔ Mini-games in multiple locations  
✔ Interactive navigation system  
✔ RPG-style progression  

---

## 🧠 How It Works

- Every mini-game rewards XP
- XP is stored in browser using `localStorage`
- Level updates automatically based on XP
- Great Hall reads and displays global progress
- Castle acts as player status hub
